## 배경

ETRI(한국전자통신연구원) 대경권연구센터 의료IT융합연구실에서 두 달 인턴십을 했습니다. 2022년 7월부터 8월까지.

맡은 과제는 PPG 신호로 혈압을 추정하는 딥러닝 모델 개발이었습니다. 커프 없이 손목 센서만으로 혈압을 재는 게 목표였습니다.

## 데이터

공개 의료 데이터셋인 MIMIC-III를 사용했습니다. 중환자실 환자 1,600명의 생체신호 데이터입니다.

8초 이상 연속된 구간을 뽑고, Peak noise랑 Flat line(신호가 평평하게 죽어있는 구간)을 제거하는 전처리를 거쳐서 최종 **101,087개** 샘플을 만들었습니다.

## 모델 구조

기존 연구들은 대부분 PPG 파형 자체만 Sequential하게 넣는 방식이었습니다. 근데 PPG에서 뽑을 수 있는 정보가 두 종류라는 걸 파악했습니다.

하나는 **파형의 형태** — 수축기 피크 높이, 이완기 트로프 위치 같은 것들.

다른 하나는 **시간 영역 특성** — BPM, 맥박 간격(IBI), 상승 시간 같은 통계값들.

이 두 가지를 병렬로 처리하는 3-Branch 구조를 설계했습니다.

```text
[PPG 파형] → Branch 1 (General CNN-LSTM) ─┐
[PPG 파형] → Branch 2 (Wide CNN-LSTM)    ─┼→ Dense → SBP 예측
[시간 특성] → Branch 3 (Time-domain DNN) ─┘
```

Branch 1, 2는 수용 영역 크기만 다르게 해서 파형의 로컬/글로벌 특성을 각각 포착하고, Branch 3은 Heartpy 라이브러리로 추출한 시간 영역 특성을 받습니다.
