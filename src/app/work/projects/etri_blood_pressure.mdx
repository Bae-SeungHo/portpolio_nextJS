## ETRI Research

2022년 여름, ETRI 대경권연구센터 의료IT융합연구실에서 두 달 인턴십을 했습니다.

과제는 **PPG 신호로 혈압을 추정하는 딥러닝 모델** 개발이었습니다.
커프(혈압계 밴드) 없이 손가락 센서만으로 혈압을 재는 게 목표였는데,
센서 데이터 자체가 정밀도가 낮고 변동성이 크기 때문에 쉽게 개발할 수 있는 프로젝트는 아니였습니다. 

저는 해당 분류 모델 개발을 위해 MIMIC-III 의료 데이터셋을 사용하였습니다.

## 3-Branch Functional Model

기존 연구들은 PPG 파형을  Sequential하게 LSTM에 넣는 게 대부분이었습니다.
근데 PPG에서 뽑을 수 있는 Time-domain 특성이 더 있다는 것을 파악했습니다.

    <Text variant="body-default-s" onBackground="neutral-weak">BPM, 맥박 간격(IBI), 상승 시간, 하강 시간 등 시간축 기반 통계 지표</Text>

이 두 가지를 하나의 Input으로 넣으면 모델이 두 특성을 독립적으로 처리하기 못해 성능의 차이가 없었기 때문에,
Branch를 총 3개로 나누었습니다.

```text
[PPG 파형] → Branch 1 (General CNN-LSTM) ─┐
[PPG 파형] → Branch 2 (Wide CNN-LSTM)    ─┼→ Dense → SBP 예측
[시간 특성] → Branch 3 (Time-domain DNN)  ─┘
```

Branch 1, 2는 Window의 크기를 다르게 설정하여 센서 데이터의 로컬/글로벌 특성을 각각 포착하고,
Branch 3은 Heartpy 라이브러리로 추출한 시간 영역 데이터를 받습니다.

시간 영역 Branch가 추가됐을 때 개선 폭이 제일 컸습니다.
파형 형태만으로는 놓치는 혈압 관련 정보가 시간 영역에 담겨있다는 의미입니다.

## Lesson & Learned

"모델 구조보다 데이터 이해가 먼저다" — 이 말이 얼마나 맞는 말인지 이 프로젝트에서 처음 제대로 느꼈던 것 같습니다.
Branch 구조를 생각해낸 것도 PPG 신호를 직접 뜯어보다 보니 발견할수 있었습니다.

2개월이라는 짧은시간 연구소에서 근무하였지만, 시계열 데이터를 입력으로 하는 모델 개발 경험을 쌓았고,
배운것을 결과로 남기기 위해 찗은 논문으로 작성해 학술대회에서 발표하였습니다.
