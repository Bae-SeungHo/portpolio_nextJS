## 프로젝트 소개

AI 영상인식 스타트업 Qisens AI의 ICT 학부인턴십 프로젝트입니다 (2022.03 — 2022.06).
위성·항공 사진 분석을 자동화하는 서비스의 핵심 모델을 개발했습니다.

## 문제

항공 영상은 촬영 각도와 조명에 따라 그림자가 심하게 발생합니다.
그림자가 있으면 객체 색상이 왜곡돼 모델 정확도가 떨어집니다.
또한 같은 종류의 객체도 항공 뷰에서는 형태가 달라 지상 데이터셋
학습 모델을 그대로 쓸 수 없었습니다.

## 내 역할

- 항공 영상 데이터 수집 및 라벨링 (주차장, 도로, 배경 3클래스)
- Semantic Segmentation 모델 학습 및 최적화 (DeepLab v3+, UNet)
- 그림자 제거 GAN 모델 적용으로 전처리 파이프라인 개선
- 선박 탐지 모델 개발 및 Bounding-Box → 좌표계 투영 알고리즘 개발
- QGIS/GDAL로 픽셀 좌표를 실제 지리 좌표(shp 파일)로 변환

## 기술 스택

`Python` `PyTorch` `TensorFlow` `QGIS` `GDAL` `OpenCV` `Linux`

## 성과

그림자 제거 전처리 적용 후 Segmentation mIoU XX% → XX%로 개선.
선박 탐지 + 좌표계 투영으로 shp 파일 자동 생성 파이프라인 완성.

## 느낀 점

데이터 품질 개선이 모델 아키텍처 변경보다 훨씬 효과적이라는 것을 배웠습니다.
그림자 제거 단계 하나를 추가하는 게 복잡한 모델 앙상블보다 좋은 결과를 냈습니다.
