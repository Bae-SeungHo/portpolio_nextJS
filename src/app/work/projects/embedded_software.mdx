## 시스템 흐름

```text
Raspberry Pi (카메라)
    ↓ TCP 소켓
Python 서버 (MediaPipe 전처리 + LSTM 추론)
    ↓
GCP 클라우드
    ↓
웹 대시보드 (양치 점수 + 미흡 부위 히트맵)
```

## 딥러닝 모델

MediaPipe로 손(21개)과 얼굴(15개) 총 **36개 Landmark 좌표**를 추출합니다. 이때 현재 프레임과 다음 프레임의 차이값을 함께 수집합니다 — 절대 위치가 아닌 움직임 패턴을 학습하기 위해서입니다.

```python
# 60프레임(약 1초) Sequential 데이터를 LSTM에 입력
model = Sequential([
    LSTM(64, return_sequences=False, input_shape=(60, 36*3)),
    Dense(32, activation='relu'),
    Dropout(0.3),
    Dense(4, activation='softmax')  # 양치 방향 4클래스
])
```

왼손잡이·오른손잡이를 구분해 각각 다른 모델로 추론합니다.
