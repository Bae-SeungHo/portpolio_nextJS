## 대회 소개

산업통상자원부 주최 제19회 임베디드 소프트웨어 경진대회 출품작입니다.
비전 센서와 딥러닝을 결합해 사용자의 양치질 습관을 실시간으로 분석하고
개선점을 알려주는 시스템을 개발했습니다.

수상: 우수상 (산업통상자원부) · 대한임베디드공학회 추계학술대회 우수논문발표상

## 문제 인식

양치질은 매일 하지만 방법이 잘못된 경우가 많습니다. 기존 스마트 칫솔은
센서 정확도 한계가 있고 가격이 비쌌습니다. 카메라 하나로 비접촉 분석이
가능하다면 더 저렴하고 정확한 솔루션이 될 것이라 생각했습니다.

## 시스템 구조

비전 센서에서 영상을 캡처해 TCP 소켓으로 Python 서버에 전송합니다.
서버에서 OpenCV로 얼굴 하관과 손 관절 특징을 추출하고,
TensorFlow 모델이 양치 방향을 분류합니다.
결과는 웹 대시보드에 미흡 부위 히트맵으로 표시됩니다.

## 내 역할

- 안면 및 손 관절 특징 추출 (OpenCV, MediaPipe)
- 데이터 전처리 및 정형화
- 딥러닝 분류 모델 구현 및 최적화 (TensorFlow)
- 임베디드 보드와 서버 간 TCP 통신 구현
- 양치 점수 및 미흡 부위 계산 알고리즘 개발

## 기술 스택

`Python` `TensorFlow` `OpenCV` `Linux` `GCP` `HTML5` `TCP Socket` `임베디드 보드`

## 성과

우수상 수상 (산업통상자원부 주관 전국 대회)

대한임베디드공학회 추계학술대회 우수논문발표상 수상

실시간 분석 레이턴시 목표치 달성 (추후 수치 업데이트 예정)

## 느낀 점

하드웨어와 소프트웨어를 동시에 다루는 경험이 처음이었습니다.
임베디드 보드의 제한된 메모리에서 OpenCV가 크래시 나는 문제를
TCP로 연산을 서버에 위임하는 방식으로 해결하면서,
아키텍처 설계의 중요성을 배웠습니다.
