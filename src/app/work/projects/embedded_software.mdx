## 프로젝트 개요

고가의 스마트 칫솔 없이 카메라 하나로 양치 습관을 분석할 수 있으면 어떨까?
라는 아이디어로 시작하게 된 "양치의 정석" 프로젝트입니다.

임베디드 보드의 Raspberry 카메라로 양치하는 손 움직임 영상을 수집하고,
LSTM 딥러닝 모델을 통해 칫솔 방향과 미흡 부위를 분석해서 웹으로 보여주는 시스템입니다.
연구실 동료 네명과 함께 만들었는데, 역할을 처음부터 잘 배분하여 원활하게 협업할 수 있었습니다.

## 시스템 흐름

```text
Raspberry Pi (카메라)
    ↓ TCP 소켓
Python 서버 (MediaPipe 전처리 + LSTM 추론)
    ↓
GCP 클라우드
    ↓
웹 대시보드 (양치 점수 + 미흡 부위 시각화)
```

## 딥러닝 모델

MediaPipe로 손(21개)과 얼굴(15개) 총 **36개 Landmark 좌표**를 추출합니다.
그리고 현재 프레임과 이전 프레임간의 Euclidean distance를 함께 수집합니다.
영상에서의 절대 위치가 아닌 사용자의 움직임 패턴을 학습하기 위해서입니다.

```python
# 60프레임(약 1초) Sequential 데이터를 LSTM에 입력
model = Sequential([
    LSTM(64, return_sequences=False, input_shape=(60, 36*3)),
    Dense(32, activation='relu'),
    Dropout(0.3),
    # + Hidden Layers
    Dense(4, activation='softmax')  # 양치 방향 4클래스
])
```

왼손잡이,오른손잡이를 포함한 다양한 환경(밝기, 거리, 얼굴 방향) 에 대해 학습하고 파인튜닝을 진행하였습니다.
