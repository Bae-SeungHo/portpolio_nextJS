## 서비스 소개

KB국민은행 AI 금융비서는 실사 아바타 AI가 고객과 직접 대화하며
금융 업무, 날씨, 취미 등 일상 대화까지 처리하는 대고객 서비스입니다.

2021년 3월 여의도 KB국민은행 신관 AI체험존에서 처음 공개되었으며,
초기 100여 개 대화 시나리오에서 출발해 지속적으로 고도화됐습니다.

[뉴스토마토 보도 (2022.06)](https://www.newstomato.com/readNewspaper.aspx?no=1126892) |
[전자신문 보도 (2024.03)](https://www.etnews.com/20240306000220)

## 내가 맡은 역할

금융AI센터에서 이 서비스의 운영·개발 담당자로 근무했습니다 (2022.12~2024.09).

**클라우드 비용 최적화**

영업 시간 외 트래픽이 거의 없는데 서버를 24시간 풀로 유지하면 비용 낭비입니다.
AWS Lambda scheduling을 적용해 영업 시간 외 파드를 자동 스케일다운하고,
HPA(Horizontal Pod Autoscaler)로 트래픽 급증 시 자동 스케일업하도록 구성했습니다.

**DevOps · CI/CD 구축**

GitHub Actions 기반 CI/CD 파이프라인을 구축했습니다.
코드 병합 시 자동으로 Docker 이미지를 빌드하고 EKS에 롤링 업데이트로 배포합니다.
무중단 배포로 서비스 안정성을 확보했습니다.

**학습 데이터 관리 및 모니터링**

대고객 봇 서비스의 대화 로그를 분석하고
학습 데이터 품질을 관리했습니다.

## 기술 스택

`Kubernetes` `AWS EKS` `Lambda` `HPA` `Docker` `CI/CD` `Python` `모니터링`

## 시스템 구조

```
고객 모바일 → AI 금융비서 (아바타 렌더링)
↓
영상합성 서버 (CPU/GPU 집약)
↓
Kubernetes EKS (HPA 자동 스케일링)
↓
Lambda scheduling (비용 최적화)

```

## 결과

클라우드 운영 비용을 의미 있게 절감했습니다.
CI/CD 자동화로 배포 프로세스가 간소화됐습니다.
서비스 중단 없는 안정적 운영 체계를 구축했습니다.
